# v3-word

DOCX Viewer with high-fidelity table rendering.

## Features
- High-fidelity table rendering (borders, padding, alignment)
- Recursive DOCX parsing supporting nested containers
- Custom Layout Engine with pagination support
- Pixel-perfect visual accuracy

## Tech Stack
- React + TypeScript + Vite
- HTML5 Canvas for rendering
- Custom DOCX Parser and Layout Engine
